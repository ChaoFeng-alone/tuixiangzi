# 推箱子
嘲风写于2018年大一上学期，参考了学长的推箱子项目

## 简介

该项目为2018年大一上学期学长课堂实战班内容，分别写了控制台版本和EGE版本，因为一些特殊原因，还写了EasyX版本（后面没有进行bug的修复、功能添加及优化）。
