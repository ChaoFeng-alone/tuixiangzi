主页-＞库函数目录-＞绘制图形相关函数-＞setfillstyle

功能：
这个函数用于设置当前填充类型。该函数的自定义填充部分尚不支持。

声明：

void setfillstyle(
    int pattern,
    color_t color,
    PIMAGE pimg = NULL
);



参数：
pattern
填充类型，可以是以下宏或值：



宏值含义


NULL_FILL1不填充


SOLID_FILL2固实填充




pupattern
color
填充颜色。

指定图案填充时的样式，目前无作用。

返回值：
（无）

示例：
设置蓝色固实填充：
setfillstyle(SOLID_FILL, BLUE);
